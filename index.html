Dataset 1: tmpbtz4x7bc.csv (Boston Building Permits)

Data Filtering Methodology
Step 1: Load Source Data
  • Building Permits: tmpbtz4x7bc.csv
  • Property Assessment: fy2025-property-assessment-data_12_30_2024.csv
Step 2: Filter Building Permits
  • worktype == "CHGOCC" — Change of Occupancy permits only
  • status == "Closed" — Completed/closed permits only
Step 3: Clean Join Keys
  • Permits: parcel_id → converted to integer string
  • Assessment: PID → converted to integer string
Step 4: Merge Datasets
  • Join type: Left join
  • Join key: parcel_id (permits) ↔ PID (assessment)
Result: 7,012 CHGOCC permits with status "Closed" → 7,012 records after merge
